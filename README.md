Приложение написано на Java с использованием стандартных Android‑компонентов и SQLite для хранения пользователей и истории заказов. После регистрации и входа пользователь попадает в каталог товаров, может открыть подробную карточку мебели, добавить позиции в корзину, выбрать способ оплаты, вариант получения (самовывоз или доставка с интервалами времени) и оформить заказ.
Каждый оформленный заказ сохраняется в локальную базу с привязкой к аккаунту: итоговая сумма, способ получения и список купленных товаров с количеством. В отдельном экране доступна история покупок пользователя, где заказы показываются в виде простого списка, что упрощает проверку работы с SQLite и навигацией между активити.
